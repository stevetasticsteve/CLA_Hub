Text features

CLAHub does a few things for texts:

    Markdown
    Timestamping
    Lexicon checing

for markdown check Markdown

Timestamps

Writing a timestamp in this format 0:00:00 will result in an automatic link to control the audio player for the text you're working on

Lexicon checking

Every night the server reads our two lexicon files (Kovol_lexicon.xslx and Kovol_verbs.xlsx) and creates a word list. CLAHub checks this word list and highligts in red any words not found. The idea is that then either spelling can be corrected or the lexicon updated.

There are some exceptions:

    Timestamps aren't checked
    Numbers aren't checked
    Anything after a bracket ( will be igored till the bracket is closed ). This allows you to write in English and not have the English highlighted as not being found in the Kovol lexicon.
